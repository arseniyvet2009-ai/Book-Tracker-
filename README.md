# Book-Tracker-
**Автор:** [Хачатрян Паргев]
**Дата** 04.05.2026
## Описание

Приложение для учёта прочитанных книг с графическим интерфейсом. Позволяет:
* Добавлять книги с указанием названия, автора, жанра и количества страниц.
* Фильтровать записи по жанру и количеству страниц.
* Сохранять и загружать данные в формате JSON.

## Инструкция по запуску

**Требования:**
* Python 3.6 или выше;
* библиотека Tkinter (обычно идёт в комплекте с Python).

**Запуск:**
1. Откройте терминал или командную строку.
2. Перейдите в директорию проекта: `cd path/to/BookTracker`.
3. Запустите приложение: `python main.py`.

## Примеры использования

**Тест 1: Добавление книги**
1. Заполните поля:
   * **Название книги:** «Мастер и Маргарита»;
   * **Автор:** «Михаил Булгаков»;
   * **Жанр:** выберите «Роман»;
   * **Страниц:** 480.
2. Нажмите **«Добавить книгу»**.
3. Убедитесь, что запись появилась в таблице.

## Особенности реализации

* **Валидация данных:** программа проверяет заполнение всех полей и корректность количества страниц.
* **Сохранение между запусками:** все книги сохраняются в `books.json`.
* **Гибкая фильтрация:** можно фильтровать одновременно по жанру и минимальному количеству страниц.
* **Интуитивный интерфейс:** все элементы управления расположены логично и понятно.
